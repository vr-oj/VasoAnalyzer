# Workspace Toolbar Inventory (Code-Derived)

## QToolBar instances
- `PrimaryToolbar` (row 1) created in `src/vasoanalyzer/ui/main_window.py:_create_primary_toolbar`, added in `src/vasoanalyzer/ui/shell/init_ui.py:init_ui`.
- `PlotToolbar` (row 2, `CustomToolbar` / `NavigationToolbar2QT`) created in `src/vasoanalyzer/ui/shell/toolbars.py:build_plot_toolbar`, attached in `src/vasoanalyzer/ui/shell/init_ui.py:init_ui`.

Notes:
- `PlotToolbar` actions differ by backend (`pyqtgraph` vs Matplotlib). Conditions are called out per row.
- Toolbars are hidden on the home screen (`show_home_screen`), visible in the analysis workspace (`show_analysis_workspace`).
